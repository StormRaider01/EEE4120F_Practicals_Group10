# EEE4120F Group 10

This is the shared github repository between Maarij Alam and Saeed Solomon for the 4th year engineering course, High Performance Embedded System (HPES) known by the course code EEE4120F.

## Repository Structure

All practicals are stored in their own dedicated folder, such as 'Practical 1' and so forth. Within each folder is the files related to the solution for said practical. These could include the analysis.m file, images taken of the output, etc.

## Practicals Overview

### Practical 1
Benchmarking manual implementations against built-in MATLAB/Octave functions.

### Practical 2
Evaluating serial vs. parallel algorithms using the Parallel Computing Toolbox (PCT).
